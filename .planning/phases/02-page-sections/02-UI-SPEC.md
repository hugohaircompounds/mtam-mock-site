## Spacing Scale

Declared values — all multiples of 4:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, gap between icon and label |
| sm | 8px | Compact element spacing, nav link gaps |
| md | 16px | Default element spacing, paragraph margin-bottom, button padding |
| lg | 24px | Card internal padding (`p-6`), section heading margin-bottom |
| xl | 32px | Layout horizontal padding (`px-8` = 32px), hero content margin-bottom |
| 2xl | 48px | Card grid gaps, major content spacing |
| 3xl | 64px | Footer vertical rhythm |
| section | 96px | Vertical padding for every content section (`py-[96px]`) |
| container | 1200px | Max-width for all section containers (`max-w-[1200px]`) |

**Exceptions:**

- Section container: `py-[96px]` (96px = 24 × 4, within scale) — used on every `<section>` element
- Accent divider line height: `h-[2px]` — decorative, not a spacing unit
- Card border radius: `rounded-[16px]` (16px = 4 × 4) — within scale
- Hero heading line-height: `leading-[1.1]` — typography ratio, not a spacing unit
- Footer py: `py-8` (32px) — intentionally lighter than section padding to signal footer boundary

## Typography

| Role | Size | Weight | Line Height | Font Family | Class Pattern |
|------|------|--------|-------------|-------------|---------------|
| Body | 16px | 400 (regular) | 1.6 | Manrope | `font-body text-[16px] leading-[1.6]` |
| Label / caption | 14px | 400 (regular) | 1.5 | Manrope | `font-body text-[14px] leading-[1.5]` |
| Section heading | 36px | 400 (regular) | 1.2 | DM Serif Display | `font-display text-[36px] leading-[1.2]` |
| Display / hero | 56px | 400 (regular) | 1.1 | DM Serif Display | `font-display text-[56px] leading-[1.1]` |

**Weight system:** Two weights only.
- Regular (400): all body copy, captions, section headings, display headings (DM Serif Display has no weight axis — it is always rendered as its default optical weight)
- Semibold (600): Manrope only — CTA button labels, card names, nav links, badge labels

**Enforcement rules (from RESEARCH.md anti-patterns):**
- No size outside these four: 14px, 16px, 36px, 56px
- No weight outside 400 and 600 — never `font-bold` (700) or `font-medium` (500)
- Hierarchy within a size created by color (`text-brand-text` vs `text-brand-muted`) and font family (display vs body), not by adding new sizes

| Destructive | `#ef4444` | `--color-error` | n/a (no Tailwind utility defined) | Not used in Phase 2 — no destructive actions |

**Accent reserved for (exact list — accent applied nowhere else):**
1. Primary CTA button background (`bg-brand-accent`) — "Start Learning" buttons in hero, pricing cards, final CTA section
2. CTA button text on light context — not applicable in this phase (all CTAs are on dark)
3. Star rating icons in testimonial cards (`text-brand-accent` on `ph-fill ph-star`)
4. Accent divider line above the manifesto block pull-quote (`bg-brand-accent h-[2px] w-16`)
5. "Most Popular" badge on the Quarterly pricing card (`bg-brand-accent text-brand-bg`)
6. Value comparison callout box border or background tint in pricing section (`border-brand-accent` or `bg-brand-accent/10`)
7. Hover state on footer links (`hover:text-brand-text`, not accent — see copywriting section)

**60/30/10 distribution by section:**
- DEFINE zone (hero, struggle, manifesto): 60% bg-brand-bg + 30% bg-brand-surface (manifesto only) + 10% accent (CTA button, divider line)
- LOCATE zone (topics, educators, founders): 60% bg-brand-bg + 30% bg-brand-surface (all cards) + 10% accent (check icons on topic cards)
- CONFIRM zone (testimonials, pricing): 60% bg-brand-surface (alternate surface for zone contrast) + 30% bg-brand-bg (section shell) + 10% accent (stars, Most Popular badge, value callout)
- EXECUTE zone (CTA, footer): 60% bg-brand-surface (CTA) + bg-brand-bg (footer) + 10% accent (CTA button)

## Animation Contract

All animations use GSAP 3.14.2 + ScrollTrigger. Zero CSS scroll-driven animations.

| Pattern | When Applied | GSAP Spec |
|---------|-------------|-----------|
| Hero immediate load | `#hero .hero-content` — fires on DOMContentLoaded, no ScrollTrigger | `fromTo: opacity 0→1, y 32→0, duration 1.0, ease power3.out, delay 0.2` |
| Single element reveal | Section headings, manifesto block, founders content, CTA content | `fromTo: opacity 0→1, y 24→0, duration 1.0, ease power2.out, scrollTrigger start: top 80%` |
| Stagger reveal (cards/items) | Struggle items, topic cards, educator cards, testimonial cards, pricing cards | `fromTo: opacity 0→1, y 40→0, duration 0.8, stagger 0.15, ease power2.out, scrollTrigger start: top 75%` |
| ScrollTrigger action | All scroll-based | `toggleActions: "play none none none"` — plays once, does not reverse on scroll-up |

All GSAP code lives in a single `<script>` block at the bottom of `<body>`, after all section HTML.
